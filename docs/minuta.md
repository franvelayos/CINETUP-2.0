Temática Desarrollo de un CINE

 Desarrollo de aplicación para un cine que pueda gestionar su cartelera y la venta de entradas.
 El objetivo es digitalizar la información de películas, funciones y usuarios, permitiendo la compra de tickets de manera eficiente.
 El sistema trabaja con personas registradas que pueden interactuar con la plataforma para consultar información y adquirir entradas.
 
Entre los servicios disponibles se incluyen:

* Consulta de cartelera de películas
* Visualización de funciones disponibles
* Compra de entradas

El sistema almacena información personal de los usuarios, incluyendo datos de identificación y
acceso.

El usuario puede realizar acciones como:
* Registrarse e iniciar sesión en el sistema
* Seleccionar películas y funciones
* Comprar entradas disponibles

Dentro de las características de la película es que cada una tiene un propio identificador único.
Posee título, director, categoría y resumen, además de incluir información multimedia. Por último,
pueden estar disponibles en cartelera o sino en la sección de próximos estrenos.

 Cada película en cartelera puede contar con una o varias funciones disponibles, las cuales a su vez
contienen una cantidad determinada de tickets. Estos mismos registran al usuario que realizó la
compra, la película y función seleccionada, numero de asiento, precio de la entrada y fecha de
comprar.

Consideraciones:
* Un usuario puede adquirir múltiples tickets
* No todos los usuarios necesariamente realizan compras
* Una película puede tener múltiples funciones asociadas
* No se pueden vender tickets para funciones no disponibles
